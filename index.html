Introduction

This site relates to freeCodeCamp's "Technical Documentation" challenge. Since I do not currently have the knowledge to write technical coding documentation, I have decided to use this project as an opportunity to catalogue my observations and learnings as I move through the user stories. The hope is that this serves as vehicle to reinforce learning and a point of reference for the future.

Headers

Starting with the basics, here are some things I have learned about headers.

  • The header tag is a relatively new HTML5 element
  • The header tag can be used to create the web page header, including logo and navigation buttons etc.
  • Not to be confused with the h1 tag which provides the heading for written content
  • Since both header and h1 are block elements, containing one within the other had undesired effects

For example, you should not structure your header as follows:


            <header>
              <h1>h1 tag in a header</h1>
            </header>
